## Тема  
Мобильное приложение и серверная часть системы «Умная парковка»  

## Цель проекта  
Создать прототип системы «Умная парковка», включающей Android-приложение и сервер-эмулятор Raspberry Pi, обеспечивающий обмен данными в реальном времени.

## План работ  
1. Оформление требований и архитектуры проекта  
2. Разработка серверной части (эмуляция Raspberry Pi с Flask)  
3. Разработка клиентской части (Android-приложение на Java)  
4. Интеграция клиента и сервера через REST API  
5. Логирование действий на сервере  
6. Тестирование и демонстрация работы системы  

## Требования  

### Сервер (Python, Flask)  
1. Эмуляция работы парковочных датчиков (занято/свободно)  
2. REST API для получения и изменения состояния парковочных мест  
3. Запись всех действий (логов и событий) в файлы  
4. Ответ клиенту в формате JSON  

### Клиент (Android, Java)  
1. Отображение текущего состояния парковочных мест  
2. Возможность «занять» и «освободить» место через интерфейс  
3. Отправка запросов на сервер и получение ответов  
4. Простая визуализация парковки (например, зелёные/красные ячейки)
